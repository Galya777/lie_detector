Mini Lie Detector Simulator

Описание

Проектът представлява симулатор на детектор на стрес/лъжата, реализиран с Arduino Uno.
Системата симулира биометрични показатели (пулс, GSR, температура) и визуализира състоянието чрез вграден LED и Serial Monitor.

Проектът е учебен и демонстрационен и не е медицински инструмент.

Функционалности

Генериране на случайни симулирани стойности за:

Пулс (BPM)

Кожна проводимост (GSR)

Температура (°C)

Изчисляване на комбиниран скор за стрес.

Визуална индикация чрез вграден LED:

HIGH = ALERT (червен LED)

LOW = NORMAL (зелено)

Печат на симулираните стойности и състоянието в Serial Monitor.

Хардуер (опционално)

За демонстрация без реални сензори е достатъчна само платката Arduino Uno и компютър с USB.
Ако искаш да използваш реални сензори, проектът може да бъде разширен с:

Pulse Sensor (PPG)

GSR Sensor

Температурен сензор DS18B20

LCD 16x2 с I2C

LED диоди и резистори

Софтуер

Arduino IDE 1.8.19 (Linux / Windows / macOS)

Кодът е съвместим с Arduino Uno, симулацията работи без физически сензори.

Използване

Отвори MiniLieDetector.ino в Arduino IDE.

Свържи Arduino Uno към компютъра чрез USB.

Избери Board: Arduino Uno и Port: ttyACM0 (Linux) или COM порт (Windows).

Компилирай и Upload (ако платката работи).

Отвори Serial Monitor (9600 baud) за наблюдение на симулацията.

LED на пин 13 ще светне при ALERT състояние.
